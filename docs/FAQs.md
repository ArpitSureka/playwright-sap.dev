# FAQs


### ❓ What about playwright commands and other playwright integrations?
All the playwright commands and integrations remain same.

### ❓ Can I use this with programing languages other than JavaScript ?
No, Although playwright supports other languages currenlty playwright-SAP only supports JS.

### ❓ Does this work with other SAP webapplications like Ariba, Concur, etc ?
No, currently other SAP webapplications like Ariba, Concur, etc are not supported.
